内网安全攻防1-域下信息搜集

Posted on 2020-03-08

内网常用命令

安装软件类
1.配置级本机安装信息
windows 下有cmd powershell 以及wmic（最好再终端运行） 
cmd 权限同常限制死所以可用pw 以及wmic 来执行搜集一些系统信息
ipconfig/all      网络配置信息
systeminfo         操作系统相关信息
查看安装的软件级版本路径命令
wmic product get name,version 
powershell 收集系统安装全部软件详细命令如下  
 Get-WmiObject Win32_Product | Format-List Caption,InstallDate,InstallSource,LocalPackage,Name,PackageCache,Vendor,Version

查询本机服务以及进程之类相关命令
1.本机服务
wmic service list brief  
本机进程
tasklist 不多说
本机进程对应路径以及pid
wmic process get processid,name,executablepath
wmic创建新进程
wmic process call create notepad
wmic process call create "C:\Program Files\Tencent\QQ\QQ.exe"
wmic 删除指定进程:
wmic process where name="notepad.exe" delete
wmic process where processid="2345" delete
tasklist
tasklit /v 查询某用户运行什么
tasklist 指定查询
tasklist |findstr chrome.exe
tasklist |findstr 1234
taskill
taskkill /f /pid 1234
计划任务
schtasks /query /fo list /v
系统开机时间
net statistics workstion

查询本机端口 windows补丁 以及共享表
netstat -ano (列出全部端口)

查询某一端口
netstat -an | findstr "3306"
查看windows补丁
systeminfo
wmic qfe get hotfixid
net share
wmic share get name,path,status

路由表以及防火墙开启某端口相关命令
route -print
arp-a
关闭防火墙
<win2003(需要管理员权限)
netsh firewall set opmpode disable
>2003(需要管理员权限)
NetSh Advfirewall set allprofiles state off
查看防火墙
<2003
netsh firewall show config
>2003
Netsh Advfirewall show allprofiles
修改防火墙入站程序通行规则（图有标出)
原作者--》https://blog.csdn.net/wang0112233/article/details/91802233
添加信任程序
<2003
netsh firewall add allowedprogram c:\nc.exe "allow nc" enable  
>2003
netsh advfirefall firewall add rule name= "pass nc" dir=in action=allow program="C:\nc.exe"
Linux添加端口规则：
单个端口：   iptables -A INPUT -p TCP --dport 1111 -j ACCEPT
端口段：       iptables -A INPUT -p TCP --dport 111:222 -j ACCEPT
拦截： iptables -A INPUT -p TCP --dport 111:222 -j REJECT
windows添加ip规则（入站）：
netsh advfirewall firewall add rule name=测试规则放行192.168.0.10 protocol=any dir=in action=allow remoteip=192.168.0.10
linux添加ip规则：
（单个ip放行）iptables -I INPUT -s 192.168.0.10 -j ACCEPT
（ip段拦截）iptables -I INPUT -s 192.168.0.0/24 -j DROP
注意：Linux每次都需要重启防火墙
windows的端口写法是111-222，linux是111:222
查看3389
REG query HKLM\SYSTEM\CurrentControlSet\Control\Terminal" "Server\WinStations\RDP-Tcp /v PortNumber
开启3389（需要管理员)
REG query HKLM\SYSTEM\CurrentControlSet\Control\Terminal" "Server\WinStations\RDP-Tcp /v PortNumber

域下机器以及用户信息搜集
net user 查询用户
net localgroup administrators（本地管理员组)
查询在线用户
query user
本机断开的会话(需要管理员)
net session
查询当前域用户
net user  xxx /domain
判断是否存在域
ipconfig /all
whoami
sysyteminfo
net time /domain（一般域内机器都是以域控机器weizhun)
查询域内计算机
net view /domain:hack
查询所有用户组
net group /domain
查询所有域成员机器
net group "domain computers" /domain
查询域密码策略
net accounts /domain

判断dns服务器和dc 是否是一台 一般内网dc都是有备份机
ipconfig /all
得到ip dns服务器 网关
反向解析 nslookup 解析域名ip地址 
得到ip dns服务器 网关
net config workstion

定位dc 以及域管理员
查看到所连接的域控的IP,名称以及所在站点信息
nltest /dsgetdc:hacke
域内用户详细信息
wmic useraccount get /all
